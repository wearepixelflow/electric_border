# ⚡ Electric Border Animations

Premium electric border animations built using **HTML**, **CSS**, and **JavaScript**.

Create stunning glowing borders for buttons, cards, containers, and modern UI components with smooth, lightweight animations.

---

## ✨ Features

- ⚡ Electric border animation
- 🌈 Neon glow effects
- 🎨 Fully customizable colors
- 📱 Responsive design
- 🚀 Lightweight & fast
- 📋 Copy-paste ready
- 💯 Beginner friendly

---


## 📂 Tech Stack
Only
- HTML5
- CSS
